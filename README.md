# 📄 Gerador de Currículo — MJR Dev

Projeto desenvolvido por **Mauricio Junior** como trabalho de faculdade — ADS 2025.

Gerador de currículo interativo com preview em tempo real e exportação em HTML.

---

## 🎨 Paleta de cores

| Cor | Hex |
|-----|-----|
| Roxo principal | `#887CFF` |
| Azul fundo | `#F1F9FF` |
| Cinza texto | `#5D5D5D` |
| Preto | `#000000` |

---

## ✅ Funcionalidades

- Formulário completo com 4 abas: Pessoal, Experiência, Educação e Extras
- Preview do currículo em tempo real
- Adicionar e remover experiências e formações dinamicamente
- Exportar currículo como arquivo `.html` pronto para usar
- Responsivo para celular, tablet e notebook
- Menu hamburger no mobile
- Imprimir como PDF pelo navegador (Ctrl+P)

---

## 🚀 Como usar

1. Clone ou baixe o repositório
2. Abra a pasta no VS Code
3. Instale a extensão **Live Server**
4. Clique com botão direito no `index.html` → **Open with Live Server**
5. Preencha seus dados e clique em **Exportar HTML**
